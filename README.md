# 💳 Credit Card Data Analysis using Power BI

## 📌 Project Overview

This project focuses on analyzing credit card customer and transaction data using Microsoft Power BI to generate business insights related to customer behavior, revenue trends, spending patterns, and profitability.

The project includes interactive dashboards built using Power BI, DAX, Power Query, and data modeling techniques.

---

## 🎯 Objectives

* Analyze customer demographics and spending behavior
* Identify high-revenue customer segments
* Track transaction and revenue trends
* Compare card category performance
* Understand customer profitability and spending patterns

---

## 📊 Dashboards

### 1️⃣ Customer Analysis Dashboard

* Revenue by Age Group & Gender
* Income Group Analysis
* Revenue by State and Region
* Education & Job-wise Revenue
* Marital Status Insights

### 2️⃣ Transaction Analysis Dashboard

* Quarterly Revenue Trends
* Card Category Performance
* Transaction Mode Analysis
* Expense Type vs Revenue
* Transaction Amount & Count Analysis

---

## 🛠️ Tools & Technologies

* Power BI
* DAX
* Power Query
* SQL
* Advanced Excel

---

## 📈 Key Insights

* High-income customers contribute the highest revenue
* Customers aged 25–35 show maximum spending behavior
* Married customers generate higher revenue contribution
* Online transactions show increasing trends
